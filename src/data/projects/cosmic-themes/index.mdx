## Overview

Cosmic Themes is a collection of polished Astro website templates designed to shorten the path from idea to launch.

## What I focused on

- Fast setup for marketing sites and portfolios
- Reusable sections and layout patterns
- Clean Tailwind styling that is easy to customize

## Outcome

This project made it easier to start new sites with a strong foundation instead of rebuilding the same basics each time.
