## Overview

The Void is a content-focused blog theme designed for creators who want a modern reading experience with useful publishing features.

## Highlights

- Tag and category support
- Newsletter-ready layouts
- A clean visual system centered on readability

## Outcome

It packages the common needs of a modern blog into a theme that is straightforward to publish and extend.
